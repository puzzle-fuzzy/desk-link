# Windows 0.1.68 诊断性能摘要

0.1.68 在现有本地诊断报告中增加“视频性能摘要”。它只读取已经写入本地的脱敏计数，不新增网络上传，也不在默认远程桌面界面显示。

报告会同时查看控制端的中继交付、视频提交解码和 Canvas 显示指标，并给出有限的排查线索：

- 视频已经进入解码但没有显示：优先检查 WebView2 解码能力。
- 中继没有持续完成帧：优先检查网络、中继会话或目标主机编码。
- 视频已经交付但显示帧率低或帧间隔过大：优先检查本机解码或 WebView2 渲染。

这些判断是诊断线索，不会替代两台 Windows 实机测试；复现卡顿后导出一份报告即可提供足够的下一步证据。
