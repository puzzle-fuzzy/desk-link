# Windows 0.1.35：远程画质档位

DeskLink 0.1.35 在远控工具栏加入流畅、均衡、清晰三档画质。切换操作通过端到端加密的控制通道发送，主机完成编码器重建后再向控制端确认实际档位。

## 档位

- 流畅：最高 1280 × 720、15 帧/秒、1.5 Mbps，适合网络不稳定或带宽较低时使用。
- 均衡：最高 1920 × 1080、20 帧/秒、2.5 Mbps，兼顾文字清晰度和带宽。
- 清晰：最高 1920 × 1080、30 帧/秒、4 Mbps，与 0.1.34 的默认编码参数一致。

主机切换失败时会保留原编码器和原档位，不会因此主动中断远控会话。切换成功会清除旧画面队列、生成新的解码配置并请求关键帧。

## 兼容性

此次扩展将端到端协议版本提升到 2。两台 Windows 电脑必须都安装 0.1.35；公网中继只转发加密数据，不需要随此版本重新部署。
