# Windows 0.1.87：高 DPI 下的原始像素显示

0.1.87 继续使用协议 10，修正控制端在 Windows 显示缩放大于 100% 时的 1:1 显示路径。

## 调整内容

- 1:1 模式根据 WebView 的 `devicePixelRatio` 设置 Canvas 的 CSS 宽高，让一个收到的远程像素对应一个物理显示像素。
- 适应模式仍由 `object-fit: contain` 在全屏视口中等比铺满，不改变常规控制体验。
- 画面小于视口时保持居中；画面大于视口时保持滚动浏览和原有输入坐标映射。

## 验收重点

在 Windows 显示缩放设置为 125% 或 150% 的控制端，选择“1:1 原始像素”，观察小字号文字和细线；切回“适应”后应恢复完整画面，不出现额外的布局偏移。
