# Windows 0.1.82：远程光标状态复用

## 目标

降低远程光标高频更新在控制端 WebView2 主线程上的短命对象分配，避免光标移动与视频绘制并行时产生额外垃圾回收压力。

## 改动

- 光标信号写入一个复用的 `{ x, y, visible }` 状态对象。
- `LatestFrameScheduler` 继续只提交每个动画帧收到的最新值，不复制该对象。
- 光标坐标换算、视口内隐藏、视口外显示和取消连接时的调度取消逻辑保持不变。

## 验证

- 新增调度器测试，确认可复用可变状态且仍提交最新值。
- 需要在两台 Windows 实机上观察快速移动远程鼠标时的光标跟手性、视频流畅度和重连后的光标状态。
