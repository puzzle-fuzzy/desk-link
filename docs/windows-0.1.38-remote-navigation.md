# Windows 0.1.38：1:1 画面浏览与全屏控制

## 用户结果

- 选择“1:1”后，可以点击“浏览画面”，再用鼠标拖动或滚轮查看超出窗口的区域；
- 浏览模式不会向远程电脑发送鼠标、滚轮和键盘输入，避免为了移动本地视野而误操作远端；
- 点击“继续控制”或在画面获得焦点后按 `Esc`，即可恢复远程输入；
- “适应”模式会自动退出浏览模式，因为完整画面已经位于当前窗口内；
- 全屏现在保留远程控制工具栏，可以继续切换屏幕、画质、缩放和浏览模式，并显示明确的“退出全屏”按钮。

## 实现边界

本地浏览只调整控制端 WebView 中远程画面的滚动位置。拖动坐标会被限制在画布可滚动范围内，切换模式时会释放已经按下的远程按键和鼠标按钮，并丢弃尚未发送的鼠标移动。

这一阶段没有修改远程输入协议、加密通道、Windows 采集或公网中继。0.1.38 可继续连接使用当前协议的被控端；双机验收仍建议两端安装同一最新版。
